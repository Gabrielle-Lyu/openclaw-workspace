### Project Overview
- **OCI GPU Scanner** - Cloud-native monitoring tool for GPU clusters on Oracle Cloud Infrastructure
- Stack: Prometheus + Grafana
- Supports: NVIDIA (A100, H100, B200, H200) and AMD (MI300X) GPUs
- Deployment: OKE daemon set or system service
- Features: Active (GPU-occupying) and passive health checks
- Includes pre-configured Grafana dashboards
- Integrates with OKE Node Problem Detector for auto-tagging failed nodes
