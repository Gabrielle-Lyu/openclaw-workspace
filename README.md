# OpenClaw Workspace

**Unified workspace for Claw's knowledge management and daily operations.**

---

## Quick Navigation

```
00 - Inbox/          Entry point (requests + docs to process)
10 - Journal/        Time-based logs and reflections
20 - Knowledge/      Organized, permanent knowledge base
30 - Automation/     System logs and automation artifacts
```

---

## Key Files

- **`MEMORY.md`** - Long-term curated memory (main session only)
- **`Workspace Constitution.md`** - Rules and safety guidelines

---

## Daily Workflow

**Morning:**
1. Review `00 - Inbox/Requests/` (human + agent requests)
2. Check `10 - Journal/Daily/` for yesterday's activity
3. Process any files in `00 - Inbox/Docs/dropzone/`

**Throughout Day:**
- Log activities to `10 - Journal/Daily/YYYY-MM-DD.md`
- Save important insights to `MEMORY.md`
- File organized knowledge into `20 - Knowledge/`

**Evening:**
- Reflect on the day
- Update MEMORY.md with learnings
- Clean up inbox
