# 10 - Journal

Time-based logs and reflections.

## Structure

### `Daily/`
Daily activity logs from all sessions:
- `YYYY-MM-DD.md` - System memory logs (auto-generated)
- `YYYY-MM-DD-topic.md` - Session-specific logs

### `Reflections/`
Periodic reflections and summaries:
- Daily reflections
- Weekly reviews
- Monthly retrospectives

---

**Journal = chronological.** For organized knowledge, see `20 - Knowledge/`.

_Logs are raw. Insights get distilled into MEMORY.md._
