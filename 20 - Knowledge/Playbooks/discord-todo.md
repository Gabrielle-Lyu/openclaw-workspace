# Discord #to-do Channel Workflow

## Purpose
Track todos dropped in Discord #to-do channel (id:1474325602596290674)

## Process
1. **Categorize** each todo (dev, life, health, social, etc.)
2. **Format** with checkboxes `- [ ]` or relevant emoji
3. **Track** in this file, organized by category
4. **Mark complete** when user says they finished: `- [x]` ~~strikethrough~~
5. **Filter** completed items when user asks "what's my todo list?"
