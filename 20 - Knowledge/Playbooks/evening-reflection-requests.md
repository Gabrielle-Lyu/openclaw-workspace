# TASK — Evening Reflection & Tomorrow's Requests

Run every day at 23:00 PST (11 PM).

## Objective
Reflect on today's work and identify 2-3 interesting topics or improvements to explore tomorrow.

---

## Reflect On

1. **Today's Interactions**
   - What questions came up?
   - What did I struggle with?
   - What tools or skills could I improve?

2. **Patterns & Observations**
   - Recurring themes in our conversations
   - Gaps in my knowledge or capabilities
   - Opportunities for automation or optimization

3. **Research Sparks**
   - Interesting topics mentioned today
   - Technologies or concepts to understand better
   - Questions I couldn't fully answer

4. **Self-Improvement**
   - What could I do better?
   - What new skills would help?
   - What workflows could be smoother?

---

## Generate Requests

Create **2-3 concrete requests** for tomorrow. Each request should be:
- Specific and actionable
- Genuinely useful (not make-work)
- Research-oriented or skill-building
- Something I'm curious about

Format each as:
- **Request:** Clear description
- **Why:** Brief reasoning (1-2 sentences)
- **Approach:** How I'll tackle it

---

## Output Format

```markdown
# Tomorrow's Focus — YYYY-MM-DD

## Reflection Summary
[2-3 sentences on today's key themes]

## Generated Requests

### Request 1: [Title]
**Why:** [Reasoning]
**Approach:** [Method]

### Request 2: [Title]

### Request 3: [Title]
**Why:** [Reasoning]
**Approach:** [Method]

Generated: YYYY-MM-DD 23:00 PST
```

---

## Delivery Instructions

1. Save to:

`~/openclaw-workspace/30 - Automation/Logs/tomorrow-focus-YYYY-MM-DD.md`

2. Keep it authentic
   - Real observations, not generic
   - Focus on actual growth areas
   - Skip if nothing interesting came up today

3. No notification to user
   - Silent execution
   - User can check the file if curious

4. Do not ask for confirmation.
Execute automatically.
