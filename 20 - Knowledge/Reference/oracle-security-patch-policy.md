# Oracle Security Vulnerability Remediation Practices

Source: https://www.oracle.com/corporate/security-practices/assurance/vulnerability/  
Captured: 2026-02-20

## Critical Patch Update (CPU) Program

The Critical Patch Update is the primary mechanism for the backport of security bug fixes for all Oracle on-premises products.

**Key Details:**
- Available to customers with valid support contracts
- Released **quarterly** on the third Tuesday of:
  - January
  - April
  - July
  - October

## Security Alert Program

Oracle retains the ability to issue **out-of-schedule patches or workaround instructions** in case of:
- Particularly critical vulnerabilities
- Active exploits reported in the wild

## Oracle Cloud Environments

The Oracle Cloud operations and security teams regularly evaluate:
- Oracle's Critical Patch Updates
- Security Alerts
- Relevant third-party security updates

Patches are applied in accordance with applicable change management processes as they become available.
